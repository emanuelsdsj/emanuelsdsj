## About me

Senior developer working remotely, currently at **IDid** dealing with payment security, PCI DSS compliance, fraud prevention. The kind of backend work that needs to actually work under pressure.

Started with C, Java and C++, went through the full-stack route, ended up deep in backend and infrastructure. Python is home base, but I've worked with enough languages to not be religious about it.

Lately spending a lot of time with LLMs and AI agents, not just using them, but understanding how they work and where they actually make sense to apply.

Published research on reinforcement learning applied to Pac-Man, which is exactly as fun as it sounds.

Left this profile quiet for too long, fixing that now. Pushing personal projects here daily, stuff I've been building on the side that never made it out of local folders.
